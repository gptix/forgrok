# Karpathy LLM Wiki Pattern

**Ingested:** 2026-05-04

Core idea: Use an LLM as the maintainer of a clean, interlinked Markdown wiki. Obsidian (or Emacs + Org-roam) acts as the IDE. Humans mostly write raw notes; the LLM compiles them into the living wiki.

**Status in ZHO:** Foundation layer established. Ready for synthesis.
