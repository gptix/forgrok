* 1.5 Determine active/default model for Hermes/Withnail (NEW)
** Quick check – currently loaded models
- [ ] Run this command:
  #+begin_src bash
  ollama ps
  #+end_src
  This shows which models are currently loaded into memory.

** Find what model Hermes/Withnail is configured to use
- [ ] Check for environment variables that might set the model:
  #+begin_src bash
  env | grep -iE 'ollama|hermes|withnail|model' | sort
  #+end_src

- [ ] Search the filesystem for config files that reference model names (especially hermes3):
  #+begin_src bash
  grep -r --include="*.yaml" --include="*.yml" --include="*.json" --include="*.conf" --include="*.sh" \
       -E 'hermes3|hermes3:8b|model.*hermes' ~ 2>/dev/null | head -30
  #+end_src

- [ ] Look for Hermes configuration directory and main config file:
  #+begin_src bash
  find ~ -type d -name "*hermes*" 2>/dev/null | head -10
  ls -la ~/.hermes 2>/dev/null || echo "No ~/.hermes directory"
  #+end_src

- [ ] If you find a config file (common names: config.yaml, settings.json, hermes.yaml, etc.), display the relevant sections:
  #+begin_src bash
  # Example - adjust path once you find the real file
  cat ~/.hermes/config.yaml 2>/dev/null | grep -A 20 -E 'model|llm|ollama' || true
  #+end_src

** Check how Hermes/Withnail is actually launched
- [ ] Look for systemd service or launch script:
  #+begin_src bash
  systemctl status hermes* withnail* 2>/dev/null || true
  cat /etc/systemd/system/hermes*.service 2>/dev/null || true
  #+end_src

- [ ] Search for launch scripts or entry points:
  #+begin_src bash
  grep -r --include="*.sh" -E 'ollama run|hermes3|withnail' ~ 2>/dev/null | grep -v ".git" | head -15
  #+end_src

** Check recent logs for model usage
- [ ] Look in common log locations:
