* Purpose

This directory contains knowledge nodes related to **Hermes** (the agent framework from Nous Research), with a particular focus on running and stabilizing Hermes on Raspberry Pi hardware (especially the Pi 5).

It includes troubleshooting notes, configuration recommendations, systemd unit best practices, and operational guidance for Hermes-based systems such as Withnail.
