* Purpose

This directory stores technical specification baselines of the Raspberry Pi 5
(hostname: hermes-pi) running Hermes Agent + Withnail (autonomous CEO bot).

* Current State

The file [[file:LATEST.org][LATEST.org]] always points to the most recent baseline.

* How Grok Should Use This

When the conversation involves any of the following topics:
- Raspberry Pi 5 hardware or configuration
- Hermes Agent (v0.14.0+)
- Withnail (CEO bot, gateway, skills, cron jobs)
- Ollama setup or models running on the Pi
- Power, thermal, networking, or resource usage on hermes-pi
- Tailscale, SSH, or air-gapped aspects of the Pi

→ You should read the file [[file:LATEST.org][LATEST.org]] (or the latest file in =baselines/=)
   to understand the current technical state before giving advice.

* Update Process

The baselines are generated by running this command on the laptop:

#+begin_src sh
collect-pi-baseline
#+end_src

This command:
1. Runs the collection script on hermes-pi via SSH
2. Downloads the result directly into =baselines/=
3. Updates =LATEST.org= (when improved)
4. Opens the file in Emacs

* Directory Structure

- =baselines/= — Timestamped historical specification reports
- =LATEST.org= — Symlink to the newest baseline (for easy reference)
- =README.org= — This file (instructions + context)
